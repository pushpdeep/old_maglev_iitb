Simulation of Control of Magnetic Levitation System
Plant Modeling
Aim

The aim of this experiment is to learn how to derive a transfer function model of the magnetic levitation (Maglev) plant. The Maglev transfer function is put in a standard second order form, and expressions are obtained for the damping factor, natural frequency, and steady state (DC) gain in terms of the physical parameters. The effect of different physical parameter values on the damping etc. is studied via a simulation program based on these expressions.

Introduction

A mathematical model of the Maglev plant enables us to study the steady state and dynamic characteristics of the plant. Once a good model is obtained, a suitable control law can be developed to improve the stability and performance of the system. We prefer to obtain a simple linear transfer function model of the Maglev plant, because it facilitates analytical design of PID controllers in the subsequent experiments.


We first develop a nonlinear mathematical model of the Maglev plant, based on fundamental laws of physics. The nonlinear model is linearized around an operating point using first order Taylor series expansion. The result is a second order linear model. The second order model is rearranged and put into the standard form of a second order transfer function. Then, the expressions for damping factor, natural frequency, and plant gain can be easily derived in terms of the physical parameters.


We shall study the effect of different values of the physical parameters on the standard second order model parameters like damping factor etc.. This is done via a simulation program based on these expressions.

Things to think about

  1. Actuators are required to supply the inputs to plants. A model of the actuator is therefore required to be developed along with that of the plant, if the model is to be useful for control design. What is the actuator for the Maglev system? Have we modeled it? If so, what is the model?

  2. The output of the Maglev plant is the height or position. A measurement sensor is required to get the position for feedback. What kind of position sensor is used for the maglev system? Have we modeled it? If so, what is the model?

  3. Is the plant gain of the maglev transfer function low or high? Will this be a problem for position measurement?
